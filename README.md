# finxos-icons-editor-frontend
> Icons for editor-frontend.

## installation
```shell
npm install afeiship/finxos-icons-editor-frontend
```

## upgrade
```shell
# directly
rm -rf node_modules/finxos-icons-editor-frontend
npm install 

# OR
npm upgrade finxos-icons-editor-frontend
```
